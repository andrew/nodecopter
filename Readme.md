# Nodecopter

A Command line utility for nodecopter

## Install

    npm install -g nodecopter

## Usage

    $ nodecopter --help

## Commands

    $ nodecopter config
    $ nodecopter demo
    $ nodecopter land
    $ nodecopter repl
    $ nodecopter reset
    $ nodecopter status
    $ nodecopter telnet
    $ nodecopter trim
